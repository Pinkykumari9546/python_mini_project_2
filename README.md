# 🎯 Perfect Guess Game

A simple and fun number guessing game built using Python. The computer randomly selects a number, and the player tries to guess it in the minimum number of attempts.

---

## 📌 Features

* Random number generation
* User-friendly input/output
* Tracks number of attempts
* Provides hints (Too high / Too low)
* Simple and beginner-friendly logic

---

## 🛠️ Technologies Used

* Python (Core Programming)
* Random Module

---

## 🚀 How to Run the Project

1. Install Python (if not already installed)
2. Clone this repository or download the file
3. Open terminal or command prompt
4. Navigate to the project folder
5. Run the game using:

```bash
python main.py
```

---

## 🎮 How to Play

1. The computer will generate a random number (usually between 1 to 100)
2. Enter your guess
3. The game will respond:

   * 📉 Too Low → Try a higher number
   * 📈 Too High → Try a lower number
4. Keep guessing until you find the correct number
5. Your total attempts will be shown at the end
